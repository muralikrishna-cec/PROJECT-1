About the Project

AI Code Review Assistant is an all-in-one tool for developers, educators, and students to review Java code intelligently. It combines traditional static analysis, flowchart visualization, plagiarism checks, and AI-driven suggestions to enhance productivity and code quality.

  • JavaParser-powered static metrics
  • Flowchart
  • AI suggestions
  • Token & Levenshtein-based plagiarism checker
